說明：手機常見失敗原因

CORS preflight 錯誤
POST 請求在手機會觸發 preflight，Google Apps Script 不回應 preflight → 失敗

解決方案
✅ 使用 GET 方式（本系統已自動採用）
✅ Apps Script 部署設為「任何人」可存取
✅ Apps Script 執行身分設為「我（部署者）」

確認 Apps Script 部署步驟：
1. 打開 Apps Script 編輯器
2. 部署 → 新增部署
3. 類型：網路應用程式
4. 執行身分：我
5. 誰可以存取：任何人
6. 複製新網址到系統設定
